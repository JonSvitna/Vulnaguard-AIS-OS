# Outreach Log

Tracks every lead contacted through cold email or LinkedIn. One row per lead. Update status as replies come in — this is the source of truth for "what's actually working" until volume justifies a real CRM (Excel Online in the Microsoft 365 environment is the next step up).

| Date sent | Lead / Company | Channel | Contact | Subject / Hook | Status | Reply notes |
|---|---|---|---|---|---|---|
| | | | | | | |

**Status values:** `sent` → `opened` (if trackable) → `replied` → `booked` / `not interested` / `no response`.

**Cadence:** add a row the day you send. Update status when something changes — don't batch it up, the value is in it staying current.
