# Phase 8: Closeout

**Objective:** Formally close the engagement, collect acceptance, issue final invoice, and capture lessons learned.

---

## Entry Criteria

- All deliverables issued to client
- Final report package acknowledged by client
- Validation phase complete (if applicable)

---

## Activities

### 8.1 Send Closeout Report
- Summarize engagement: what was done, what was delivered, current open item status
- Include a list of all deliverables and their delivery dates
- Note any items the client accepted as risk

### 8.2 Collect Signed Acceptance Form
- Send the client acceptance form alongside the closeout report
- Request signature within 5 business days
- File the signed form in `06-closeout/`
- Do not issue the final invoice until the acceptance form is received

### 8.3 Issue Final Invoice
- Per SOW payment schedule
- Reference the engagement name and deliverable list on the invoice

### 8.4 Internal Debrief
- What went well?
- What would we do differently?
- Any process gaps to fix in the playbook?
- Any upsell / follow-on opportunities identified?

### 8.5 Archive Engagement Folder
- Confirm all files are in the correct subfolder structure
- Verify naming conventions applied to all final deliverables
- Move engagement folder to archive if using a live project management system

### 8.6 Sentinel Onboarding Offer
- If the client is not already on Sentinel CMMC, offer onboarding as a follow-on
- Highlight how findings and POA&M from this engagement map directly into Sentinel
- Refer to [Phase 9 / Sentinel Implementation runbook](09-sentinel-onboarding.md)

---

## Outputs

- Closeout report
- Signed client acceptance form
- Final invoice

- Archived engagement folder

## Common Issues

**Client won't sign the acceptance form:** Call them. Understand the objection. If there is a genuine unresolved deliverable issue, address it. If they are simply slow, remind them that the final invoice is contingent on acceptance.

**Final invoice disputed:** Refer to the SOW payment schedule. If a scope change occurred mid-engagement that wasn't documented with a change order, this is a hard lesson — document change orders in real time.

**Lessons learned never captured:** Block 30 minutes internal immediately after closeout call. Don't skip this step — it's how the playbook improves.
