# About Me

Solo developer / solo entrepreneur. Background spans hacking (CompTIA PenTest+) and information security, with a Master's in Computer Forensics and a BS in Computer Networking. Builds and ships a wide range of applications — some pure learning projects, some practical tools — and treats the work as both a craft and a way to keep learning new technology.

Primary creative craft is **SaaS products**, not agentic flows — comfortable with and uses both, but SaaS is what energizes the build. Agents/automation get reached for specifically to take workload off as a one-person team.

Biggest recurring drains: lead analysis/scraping, and content creation + posting for social media — the latter is disliked more than any technical work, dev side included.
